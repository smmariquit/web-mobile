# Contributing to web-mobile

Thanks for helping improve this freelance portfolio site.

## Report issues

Open a [GitHub issue](https://github.com/smmariquit/web-mobile/issues) for broken links, layout bugs, or copy fixes. Include browser, viewport width, and a screenshot when helpful.

## Developers

### Setup

```sh
bun install
bun dev       # Nuxt dev server
```

### Pull requests

1. Branch from **`main`**.
2. Use [Conventional Commits](https://www.conventionalcommits.org/) (`fix(ui): …`, `feat: …`).
3. Run `bun run build` before requesting review for substantive changes.
4. Do not commit secrets (`.env`, worker tokens, contact API keys).
